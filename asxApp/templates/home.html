Welcome to ASX_DB


The purpose of this application is to allow users to track and view data about companies listed on the Australian Stock Exchange (ASX).

Use the links in the navigation bar above to explore the site:

  • Users: View an index of all registered users
  • Tickers: View an index of companies that we are tracking
  • Login/Register: Create or login to your ASX_DB account

This application demonstrates CRUD web development capabilities with Python using the Flask framework in combination with:
  • postgresql
  • psycopg2
  • jinja
  • marshmallow
